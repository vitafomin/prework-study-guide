CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS: Style an element directly in the HTML file
  • Internal Style Sheet: Embeds an entire stylesheet directly in the HTML file using a "style" element in the "head" element. Contains rules for the webpage in the head section of the HTML
  • External CSS Style Sheet: Keeps all CSS rules in a seperate file
  • When using an external style sheet, we link the file using a "link" element placed in the HTML's "head"
  • CSS Rule; Declarations: contain two components: - The CSS property we want to apply the value of the property - Property: Color Value: Blue
  • Margin: Indicates how much space we want around the outside of an element
  • Padding: Adds space around the content inside an element
  • *: Applies a rule to all the elements visible on the page
